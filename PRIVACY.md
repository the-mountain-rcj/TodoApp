# Privacy

TodoApp is a local-first offline todo app.

- Todo data is stored on the user's computer at `%APPDATA%\TodoApp\todos.json`.
- Holiday calendar data may be fetched from public holiday APIs and cached at `%APPDATA%\TodoApp\holidays`.
- TodoApp does not provide accounts, cloud sync, telemetry, analytics, or advertising in this version.
- Uninstalling the app removes installed program files. User data under `%APPDATA%\TodoApp` is intentionally kept so reinstalling or upgrading does not delete todos.
